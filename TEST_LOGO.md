# 🎨 Швидке тестування нового логотипу

## Відкрийте сайт

1. **Запустіть демо-сервер:**
   ```powershell
   python -m http.server 3000
   ```

2. **Відкрийте в браузері:**
   ```
   http://localhost:3000/nrg-index.html
   ```

## Що перевірити

### ✅ Header (верх сторінки)
- [ ] Логотип відображається у лівому верхньому куті
- [ ] Це новий логотип з молекулярною структурою
- [ ] Текст "NRG-UA SUPPLY PARTNER" читається
- [ ] При наведенні логотип злегка збільшується

### ✅ Footer (низ сторінки)
- [ ] Логотип відображається в футері
- [ ] Логотип менший за розміром (40px)

### ✅ Favicon (іконка браузера)
- [ ] У вкладці браузера відображається іконка з молекулою
- [ ] Іконка синього кольору

### ✅ Інші сторінки
Перевірте логотип на:
- http://localhost:3000/pages/nrg-about.html
- http://localhost:3000/pages/nrg-brands.html
- http://localhost:3000/pages/nrg-catalog.html
- http://localhost:3000/pages/nrg-contacts.html

### ✅ Адаптивність
1. Зменшіть вікно браузера
2. Логотип має зменшуватися на малих екранах
3. Відкрийте Developer Tools (F12)
4. Перемкніться на мобільний вигляд
5. Логотип має бути 35px висоти

## Якщо щось не так

### Логотип не відображається
```powershell
# Перевірте наявність файлів
ls images/
```

Повинно бути:
- favicon.svg
- nrg-logo.svg

### Відображається старий логотип
```powershell
# Очистіть кеш браузера
Ctrl + F5  # або Ctrl + Shift + R
```

### Потрібно змінити розмір
Відредагуйте HTML файли, змінивши `height="50"` на потрібне значення.
