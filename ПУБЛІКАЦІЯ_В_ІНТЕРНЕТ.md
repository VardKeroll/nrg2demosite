# 🌐 Публікація проекту в Інтернет

## 📋 Доступні варіанти хостингу

### 1. ⭐ GitHub Pages (РЕКОМЕНДОВАНО - БЕЗКОШТОВНО)

**Переваги:**
- ✅ Повністю безкоштовно
- ✅ Автоматичний деплой з Git
- ✅ HTTPS за замовчуванням
- ✅ Швидко і надійно

**Як опублікувати:**
1. Запустіть `PUBLISH-TO-GITHUB.bat`
2. Або читайте детальну інструкцію в `GITHUB_PAGES.md`

**URL сайту:**
```
https://YOUR_USERNAME.github.io/nrg-ua-energy/nrg-index.html
```

**Обмеження:**
- ❌ Backend (Python/FastAPI) не працює
- ✅ Frontend (HTML/CSS/JS) працює ідеально


---

### 2. 🚀 Netlify (БЕЗКОШТОВНО)

**Переваги:**
- ✅ Безкоштовний план
- ✅ Автоматичний деплой
- ✅ Власний домен можливий
- ✅ Форми працюють

**Як опублікувати:**
1. Зареєструйтесь на https://netlify.com
2. "Add new site" → "Import from Git"
3. Підключіть GitHub репозиторій
4. Build settings: Leave default
5. Deploy!

**URL сайту:**
```
https://nrg-energy.netlify.app/nrg-index.html
```

---

### 3. 🔥 Vercel (БЕЗКОШТОВНО)

**Переваги:**
- ✅ Безкоштовний план
- ✅ Дуже швидкий
- ✅ Автоматичний деплой

**Як опублікувати:**
1. Зареєструйтесь на https://vercel.com
2. "New Project"
3. Імпортуйте з GitHub
4. Deploy

**URL сайту:**
```
https://nrg-energy.vercel.app/nrg-index.html
```

---

### 4. 🐍 Backend (Python FastAPI)

Для запуску Backend потрібен окремий хостинг:

#### A) Railway.app (РЕКОМЕНДОВАНО)

## 📝 Швидкі інструкції

### GitHub Pages (найпростіше):

```bash
# 1. Запустіть скрипт
PUBLISH-TO-GITHUB.bat

# 2. Створіть репозиторій на GitHub
# 3. Виконайте команди з інструкції
# 4. Settings → Pages → Source: GitHub Actions
# 5. Готово! Сайт онлайн через 2-5 хвилин
```

### Netlify:

```bash
# 1. Push код на GitHub (PUBLISH-TO-GITHUB.bat)
# 2. Зайдіть на netlify.com
# 3. "Add new site" → "Import from Git"
# 4. Оберіть репозиторій
# 5. Deploy site
```

## ⚠️ Важливо пам'ятати

1. **GitHub Pages:**
   - Тільки статичні файли (HTML, CSS, JS)
   - Backend не працює
   - Безкоштовно для публічних репозиторіїв

2. **Backend хостинг:**
   - Потрібен для Python/FastAPI
   - Railway, Render, тощо
   - Оновіть API URL у frontend коді

3. **Оновлення:**
   ```bash
   git add .
   git commit -m "Update"
   git push
   # GitHub Pages оновиться автоматично
   ```
