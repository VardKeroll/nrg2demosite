## 🎯 ЯК ЦЕ ПРАЦЮЄ

### Для користувача (1 дія):
```
Подвійний клік START-DEMO.bat
```

### Що відбувається автоматично:
1. ✅ Перевірка наявності Python
2. ✅ Якщо немає → встановлення Python 3.11 (winget або прямо з python.org)
3. ✅ Створення віртуального середовища (venv/)
4. ✅ Оновлення pip
5. ✅ Встановлення всіх залежностей з requirements.txt
6. ✅ Запуск HTTP сервера на порту 3000
7. ✅ Автоматичне відкриття браузера

**Час**: Перший запуск ~5 хв, наступні ~5 сек
