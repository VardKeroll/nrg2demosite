# NRG-UA — B2B Дистриб'ютор

## 🚀 Швидкий старт (Демо-режим)

> **✨ Працює БЕЗ VS Code! Просто подвійний клік на .bat файл**

### ⚡ Автоматичний запуск (Windows):
1. Відкрийте папку проекту в провіднику
2. Двічі клікніть на **`START-DEMO.bat`**
3. Python та всі залежності встановляться **автоматично** ✨
4. Сайт відкриється в браузері

**Увага**: При першому запуску може знадобитись до 5 хвилин для завантаження та установки Python.

### 🔧 Додаткові команди:
```cmd
SETUP.bat          - Тільки налаштування середовища
START-BACKEND.bat  - Запуск Backend API на порту 8000
```

Детальніше: [SETUP.md](SETUP.md) | [QUICKSTART.txt](QUICKSTART.txt)

### Вручну (будь-яка ОС):
```bash
# Відкрийте термінал в папці проекту
python -m http.server 3000

# Відкрийте в браузері:
# http://localhost:3000/nrg-index.html
```

## 🐍 Python & Технічні вимоги

### Автоматична установка
Проект налаштований для **автоматичної установки Python 3.11** при першому запуску.

Файл `auto-setup.ps1` автоматично:
- ✅ Встановлює Python 3.11 (якщо не встановлений)
- ✅ Створює віртуальне середовище
- ✅ Встановлює всі залежності з `backend/requirements.txt`
- ✅ Налаштовує VS Code (якщо використовується)

### Вимоги до Python
- **Версія**: Python 3.11.x або 3.12.x
- **Рекомендовано**: Python 3.11.0

### Backend залежності (встановлюються автоматично)
- FastAPI 0.109.0 - веб фреймворк
- Uvicorn 0.27.0 - ASGI сервер  
- SQLAlchemy 2.0.25 - ORM
- Pydantic 2.5.3 - валідація даних
- та інші (див. `backend/requirements.txt`)

**Детальна інструкція**: [SETUP.md](SETUP.md)

## 🖥️ Повний функціонал (з бекендом)

Для повної роботи системи потрібно:

1. Встановіть залежності:
```bash
cd backend
pip install -r requirements.txt
```

2. Запустіть бекенд:
```bash
python -m uvicorn main:app --reload --port 8000
```

3. Запустіть фронтенд:
```bash
# В кореневій папці
python -m http.server 3000
```
