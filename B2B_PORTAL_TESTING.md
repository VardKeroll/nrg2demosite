# Тестування кнопки B2B Portal

## Опис функціоналу

Кнопка **B2B Portal** з'являється в header сайту замість кнопки "Увійти" після авторизації користувача.

### Особливості:
- 🔗 Посилання: https://nrgportal.com.ua
- 🔒 Доступна тільки для авторизованих клієнтів
- 🌍 Відкривається в новій вкладці
- 💬 При наведенні показує підказку: "Тільки для зареєстрованих клієнтів" (UA) / "For registered clients only" (EN)
- 🎨 Стиль: primary button з іконкою кошика

---

## Як протестувати

### Варіант 1: Через сторінку особистого кабінету

1. Відкрийте сайт
2. Натисніть кнопку "Увійти" в header
3. Авторизуйтеся:
   - **Email**: `partner@test.com`
   - **Password**: `partner123`
4. Після успішного входу кнопка "Увійти" автоматично зміниться на "B2B Portal"

### Варіант 2: Через консоль браузера (для швидкого тестування)

1. Відкрийте консоль браузера (F12 → Console)
2. Виконайте команди:

**Показати кнопку B2B Portal:**
```javascript
localStorage.setItem('isAuthenticated', 'true');
window.dispatchEvent(new CustomEvent('userLoggedIn'));
```

**Приховати кнопку B2B Portal (повернутися до кнопки "Увійти"):**
```javascript
localStorage.removeItem('isAuthenticated');
window.dispatchEvent(new CustomEvent('userLoggedOut'));
```

**Або просто перезавантажте сторінку:**
```javascript
location.reload();
```

## Перевірка функціоналу

- ✅ Кнопка "Увійти" видима для неавторизованих користувачів
- ✅ Кнопка "B2B Portal" видима для авторизованих користувачів
- ✅ При натисканні на "B2B Portal" відкривається https://nrgportal.com.ua в новій вкладці
- ✅ Підказка відображається при наведенні
- ✅ Переклад працює (UA/EN)
- ✅ Статус авторизації зберігається після перезавантаження сторінки
- ✅ При logout кнопка знову змінюється на "Увійти"

## Технічні деталі

### Файли змінено:
- `nrg-index.html` - додано HTML для обох кнопок
- `js/nrg-main.js` - додано клас `AuthManager`
- `pages/nrg-cabinet.js` - додано тригери подій авторизації
- `data/i18n/uk.json` - додано переклад українською
- `data/i18n/en.json` - додано переклад англійською

### Використовувані події:
- `userLoggedIn` - тригериться при успішному вході
- `userLoggedOut` - тригериться при виході
- `storage` - відстежує зміни в localStorage

### LocalStorage ключі:
- `isAuthenticated` - статус авторизації (boolean)
- `nrg_partner_token` - токен авторизації
