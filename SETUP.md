# 🚀 Автоматичне налаштування середовища NRG-UA Energy

> **✨ Працює БЕЗ VS Code або будь-якого IDE!**  
> Просто подвійний клік на `.bat` файл - все інше автоматично.

Цей проект налаштований для **автоматичної установки Python** та всіх необхідних залежностей без потреби у встановленні додаткового ПЗ.

## ⚡ Швидкий старт (для перевіряючих)

### Варіант 1: Автоматичне налаштування + запуск демо

Просто відкрийте папку проекту в провіднику Windows та двічі клікніть:
```
START-DEMO.bat
```

Скрипт автоматично:
- ✅ Перевірить наявність Python
- ✅ Встановить Python 3.11, якщо його немає
- ✅ Створить віртуальне середовище (venv/)
- ✅ Встановить усі залежності
- ✅ Запустить демо сервер
- ✅ Відкриє сайт у браузері

**VS Code НЕ ПОТРІБЕН!** Все працює через подвійний клік .bat файлів.

### Варіант 2: Запуск Backend API

Для запуску серверної частини (FastAPI):
```
START-BACKEND.bat
```

API буде доступне на:
- http://localhost:8000  
- http://localhost:8000/docs (Swagger документація)

### Варіант 3: Тільки налаштування середовища

Якщо потрібно тільки налаштувати Python без запуску:
```
SETUP.bat
```

## 🔧 Ручне налаштування (якщо автоматика не спрацювала)

### Крок 1: Встановіть Python
Завантажте Python 3.11 з офіційного сайту:
https://www.python.org/downloads/

**ВАЖЛИВО**: Під час встановлення поставте галочку:
☑️ Add Python to PATH

### Крок 2: Створіть віртуальне середовище
```cmd
python -m venv venv
```

### Крок 3: Активуйте віртуальне середовище
```cmd
venv\Scripts\activate
```

### Крок 4: Встановіть залежності
```cmd
pip install -r backend\requirements.txt
```

### Крок 5: Запустіть демо
```cmd
python -m http.server 3000
```

Відкрийте в браузері: http://localhost:3000/nrg-index.html

## 🎯 Для розробників

### Запуск backend API
```cmd
cd backend
..\venv\Scripts\activate
uvicorn main:app --reload
```

API буде доступне на: http://localhost:8000
Документація API: http://localhost:8000/docs

## ❓ Часті питання

### Скрипт не може встановити Python автоматично
1. Перевірте інтернет з'єднання
2. Спробуйте запустити PowerShell від імені адміністратора
3. Встановіть Python вручну (див. розділ "Ручне налаштування")

### Python встановлений, але система його не бачить
1. Перезапустіть термінал/командний рядок
2. Перезавантажте комп'ютер
3. Перевірте що Python додано до PATH

### Конфлікт версій Python
Якщо у вас встановлено декілька версій Python:
1. Використовуйте `py -3.11` замість `python`
2. Або в налаштуваннях VS Code вкажіть шлях до потрібного Python

### Помилки при встановленні пакетів
Оновіть pip:
```cmd
python -m pip install --upgrade pip
```

## 📞 Підтримка

Якщо виникли проблеми з налаштуванням:
1. Перевірте файл `.vscode/settings.json` - там вказано рекомендовані налаштування
2. Видаліть папку `venv` та запустіть `SETUP.bat` заново
3. Перевірте логи в терміналі - там буде детальна інформація про помилки
