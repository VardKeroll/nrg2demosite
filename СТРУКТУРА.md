
## 🔄 Життєвий цикл файлів

### При першому запуску:
1. Користувач: Подвійний клік `START-DEMO.bat`
2. Система: Запускає `auto-setup.ps1`
3. Скрипт: Завантажує Python 3.11
4. Скрипт: Створює `venv/`
5. Скрипт: Встановлює пакети з `backend/requirements.txt`
6. Система: Запускає `python -m http.server 3000`
7. Браузер: Відкриває `nrg-index.html`

### При наступних запусках:
1. Користувач: Подвійний клік `START-DEMO.bat`
2. Система: Використовує існуючий `venv/Scripts/python.exe`
3. Система: Запускає сервер (~5 секунд)

## 🎓 Навчальні матеріали

### Для початківців:
1. Прочитайте `ЧИТАЙТЕ_МЕНЕ.txt`
2. Запустіть `START-DEMO.bat`
3. Дочекайтесь відкриття браузера
4. Готово! 🎉

### Для досвідчених:
1. Перегляньте `pyproject.toml` - залежності
2. Вивчіть `backend/main.py` - API endpoints
3. Налаштуйте `.env` для production
4. Запустіть через `uvicorn` напряму
