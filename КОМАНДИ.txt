🚀 ОСНОВНІ КОМАНДИ (подвійний клік):
───────────────────────────────────────────────────────────────────

START-DEMO.bat
   ✓ Запускає демо сайт на http://localhost:3000
   ✓ Автоматично встановлює Python якщо потрібно
   ✓ Відкриває браузер автоматично
   ⏱️  Перший запуск: ~5 хвилин | Наступні: ~5 секунд

START-BACKEND.bat
   ✓ Запускає Backend API на http://localhost:8000
   ✓ Swagger документація: http://localhost:8000/docs
   ✓ Використовує FastAPI + Uvicorn

SETUP.bat
   ✓ Налаштовує Python та віртуальне середовище
   ✓ Встановлює всі залежності
   ✓ НЕ запускає сервери

CHECK-PYTHON.bat

🎯 ТИПОВІ СЦЕНАРІЇ:
───────────────────────────────────────────────────────────────────

Сценарій 1: Перший запуск на новому комп'ютері
   1. Двічі клікніть START-DEMO.bat
   2. Почекайте ~5 хвилин
   3. Сайт відкриється автоматично

Сценарій 2: Розробка (щоденна робота)
   1. Двічі клікніть START-DEMO.bat (для frontend)
   2. Двічі клікніть START-BACKEND.bat (для API)
   3. Працюйте!

Сценарій 3: Щось не працює
   1. Запустіть TEST-SETUP.bat
   2. Подивіться що не так
   3. Запустіть SETUP.bat для відновлення
   4. Якщо проблеми залишились - див. ІНСТРУКЦІЯ.txt

Сценарій 4: Перевірка після налаштування
   1. Запустіть CHECK-PYTHON.bat
   2. Переконайтесь що всі пакети встановлені
   3. Готово!

💡 ШВИДКІ ПІДКАЗКИ:
───────────────────────────────────────────────────────────────────

• VS Code НЕ потрібен - все працює через .bat файли
• Python НЕ потрібно встановлювати вручну
• При першому запуску потрібен інтернет
• Віртуальне середовище створюється автоматично в venv/
• Для зупинки сервера: Ctrl+C або закрийте вікно


❓ ЧАСТІ ПИТАННЯ:
───────────────────────────────────────────────────────────────────

Q: Скільки займає перший запуск?
A: ~3-5 хвилин (завантаження Python + встановлення пакетів)

Q: А наступні запуски?
A: ~5 секунд (все вже налаштовано)

Q: Потрібні права адміністратора?
A: Бажано для установки Python, але не критично

Q: Працює без інтернету?
A: Так, якщо Python вже встановлений (тільки перший раз потрібен)

Q: Можна видалити venv/ та почати заново?
A: Так! Просто видаліть папку venv/ і запустіть SETUP.bat

Q: Де переглянути логи помилок?
A: В консолі (чорне вікно) після запуску .bat файлу
